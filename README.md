# CHERISH 飾品電商
<img src="readmeImage/封面.png" alt="封面">

## 專案介紹
[Demo Website](https://hsuan890125.github.io/CHERISH/#/)

以飾品電商為主題的 SPA 網站，包含 RWD 設計，而網頁整體以簡約風格為主。

### 使用技術
- `SCSS`
- `Vue 3`
- `Vue Cli`
- `Vue Router`
- `Vue Axios`
- `vue-loading-overlay`
- `Vee Validation`
- `RWD`
- `Bootstrap 5`
